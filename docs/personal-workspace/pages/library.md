# Personal Page Concept: Library

Updated: 2026-05-18

## Purpose

I am thinking about Library as the user's personal content inventory.

It should help users find, reuse, organize, and repurpose the work they have already created.

## What It Should Hold

Library should include:

- Draft posts.
- Scheduled posts.
- Published posts.
- Generated media.
- Uploaded media.
- Templates.
- Content pillars.

## Main User Questions

Library should answer:

- What have I already created?
- Which drafts still need work?
- Which assets can I reuse?
- Which templates can help me start faster?
- Which post should I repurpose?

## Ideal Flow

The user opens Library, filters or searches for an item, and then chooses what to do with it.

They might:

- Schedule a draft.
- Repurpose a published post.
- Use a template.
- Upload media.
- Send an item back to Generate.

## Success Outcome

Library succeeds if past work becomes reusable instead of forgotten.
