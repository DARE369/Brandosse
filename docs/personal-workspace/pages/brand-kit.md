# Personal Page Concept: Brand Kit

Updated: 2026-05-18

## Purpose

I am thinking about Brand Kit as the user's personal brand memory.

It should help content sound and look consistent without the user repeating brand instructions every time they generate content.

## What It Should Capture

Brand Kit should capture:

- Brand name.
- Brand voice.
- Audience.
- Visual style.
- Colors.
- Positioning.
- Things to avoid.
- Brand assets.

## Main User Questions

Brand Kit should answer:

- What should my content sound like?
- What visual direction should my content follow?
- What should the system remember about my brand?
- What assets should be reused?

## Ideal Flow

The user sets up a Brand Kit manually or through a guided flow. Once saved, it influences future generation and post preparation.

## Success Outcome

Brand Kit succeeds if generated content feels more aligned with the user's brand by default.
